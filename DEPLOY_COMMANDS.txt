# ========================================
# SIMPLE DEPLOYMENT COMMANDS
# Copy and paste these one by one
# ========================================

# STEP 1: Authenticate with GitHub (run this first)
# This will open your browser
gh auth login

# When prompted:
# - Choose: GitHub.com
# - Choose: HTTPS
# - Choose: Login with a web browser
# - Copy the code shown and paste in browser

# ========================================

# STEP 2: Create the repository on GitHub
gh repo create HomeLoanVsODIndiaOct2025 --public --description "Comprehensive Home Loan Comparison Tool for India - Regular EMI vs Overdraft (SBI MaxGain, ICICI). Real bank data Oct 2025, tax calculator, prepayment simulator, hidden issues & smart strategies." --source=. --remote=origin --push

# ========================================

# THAT'S IT! The above command:
# - Creates the GitHub repository
# - Makes it public
# - Adds description
# - Sets up remote
# - Pushes all your code
# - All in ONE command!

# ========================================

# STEP 3: Get your repository URL
gh repo view --web

# This will open your GitHub repository in browser

# ========================================
